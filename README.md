# Лабораторная работа 2

Вариант 1: решить линейную систему `A * X = B` с точностью `0.01` методом простых итераций.

## Файлы

- `main.py` — решает систему, выводит таблицу итераций, проверяет сходимость и записывает сгенерированные артефакты.
- `report.tex` — исходник отчёта на русском языке для Overleaf или локального TeX-дистрибутива.
- `assets/program_output.png` — скриншот вывода программы (сгенерированное изображение).
- `artifacts/console_output.txt` — сгенерированный консольный вывод.
- `artifacts/iteration_table.md` — сгенерированная таблица итераций в Markdown.
- `artifacts/iteration_table.tex` — сгенерированная LaTeX-таблица, подключаемая в `report.tex`.
- `artifacts/results.json` — сгенерированные структурированные результаты.

## Запуск

```bash
python3 main.py
```

## Сборка отчёта

Установите TeX-дистрибутив, затем выполните одну из команд:

```bash
latexmk -pdf report.tex
pdflatex report.tex
```
